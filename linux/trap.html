shell中使用trap来捕获信号

Table of Contents

  • 概述
  • 使用

概述

在一个shell中执行whie循环的时候，会遇到执行ctrl-c无法中断的情况，而使用kill -9可以退出。为了避免这种情况，使用trap来捕捉SIGINT信号(ctrl-c 就是发送了SIGINT).

使用

trap 'commond-list' signal

其中signal是要捕获的信号，可以通过kill -l来查看本机可捕获的信号，如:

如:在写while循环前加上这么一段代码: 

trap 'exit 0' SIGINT

即可在键入ctrl-c正常退出
