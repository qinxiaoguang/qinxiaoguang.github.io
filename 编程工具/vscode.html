快捷键

官方快捷键手册:https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf <c>=command <s>=shift

name des
<c><s>p vscode 命令行
<c>p 跳转文件
<c><s>n new window
<c>n new file
<c>, user settings
<c>k<c>s 显示快捷键面板
<c>o open file
<ctrl>TAB open next
<ctrl><c>t toggle full screen
<c>=/- 放大/缩小
<c>b 展示/隐藏侧边栏
<c><s>e 侧边栏展示项目文件
<c><s>d show debug
<c><s>u show output panel
<c>k z zen mode
<c>\ side by side
<c><s>l 在相同字符处添加cursor
<c>D 选中下一个相似curosor
<c>k <c>f 选中内容format
